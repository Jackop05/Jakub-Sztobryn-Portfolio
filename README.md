## Installation

If you want to run the website locally:

1. **Clone the repository**:
    ```bash
    git clone https://www.jakubsztobryn.pl/
    ```
2. **Navigate to the project directory**:
    ```bash
    cd jakubsztobryn.pl
    ```
3. **Run the project**:
    Open the `index.html` file in your browser to view the website.
